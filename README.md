# weather.app
use the [rabidApis](https://rapidapi.com/community/api/open-weather-map) APIs

map from [pigeon-maps](https://pigeon-maps.js.org/)

### Features!

- choose weather forecast from map

- display temperature in Celsius or Fahrenheit
